3_3_1:tagが99.9の環境の作り方

3_3_2:buildした後に、image_test_dockerという名称のイメージを作る方法
docker run --name image_test_docker -it -d test_docker:99.9
3_3_3:イメージが作られているかの確認
docker images
3_3_4:実際にコンテナを起動する
docker exec -it image_test_docker bash
